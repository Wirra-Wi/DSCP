# 資料處理期末專案: 氣候資料視覺化工具

**作者：** wit           **領域：** data processing/visualize, spatial information technology and climatology

## 1. 專案目標

本專案旨在幫助使用者從**中央氣象局** 獲取氣候數據，並將其視覺化為地理教科書中常見的月份尺度平均氣候圖，包含氣溫與降水量


## 2. 操作說明

### 準備工作

* **環境需求：** 需安裝 `csv`, `urllib.parse`, `requests`, `BeautifulSoup`, `pandas`, `matplotlib` 等 Python 函式庫
* **檔案配置：** 程式執行檔 (`main.ipynb`) 必須與測站資料檔 (`station.csv`) 放置於同一個路徑下


### 執行步驟
1. 
**匯入函式庫與讀取資料：** 程式會自動讀取測站清單

2. **選取測站與爬取資料：**
* 使用者可輸入**測站編號 (Station ID)** 或**測站名稱**進行查詢
* 程式將透過網路爬蟲技術，從氣象局歷史資料查詢系統獲取溫度與降水數據

註：由於部分測站名稱可能重複，建議使用測站編號以確保準確性 。此外，編號開頭為 "C1" 的測站可能不提供氣溫資料

3. **產生氣候圖：**
* 程式會生成一張包含折線圖（氣溫）與長條圖（降水量）的平均氣候圖

4. **輸出 CSV 檔案：**
* 使用者可選擇是否輸出 CSV 檔案

* 程式會生成兩份文件：
`[測站名稱]氣候資料.csv`：每日詳細數據
`[測站名稱]逐月平均氣候資料.csv`：每月平均數據
